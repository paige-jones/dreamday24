# dreamday24

Install the Todo Tree plugin and put this in each students settings.json in VS Code:
    
    "todo-tree.highlights.defaultHighlight": {
        "icon": "alert",
        "type": "text-and-comment",
        "foreground": "#FFFF00",
        "background": "#FFFF00",
        "opacity": 50,
        "iconColour": "#FFFF00",
        "gutterIcon": true,
    },
